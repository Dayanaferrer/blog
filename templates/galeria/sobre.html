Como surgiu este projeto?

Este projeto surgiu no Bootcamp de Python e Django realizado pela WoMakersCode.

Requisitos do Projeto

Ambiente de Desenvolvimento: Configuração de um ambiente virtual Django local.

Controle de Versão com Git: Utilização do GitHub para controle de versão e registro claro de alterações.

Design Responsivo: Desenvolvimento de um design responsivo usando HTML5, CSS3 e Bootstrap.

Banco de Dados: Configuração de um banco de dados SQLite ou PostgreSQL para armazenamento das informações.

Modelos Django: Definição de modelos Django para representar as postagens do blog.

Funcionalidades do Blog

Página Inicial: Exibição de todas as postagens em ordem cronológica.

Páginas de Detalhes: Visualização individual das postagens.

Gerenciamento de Postagens: Capacidade de editar e excluir postagens.

Barra de Pesquisa: Busca por palavras-chave para facilitar a navegação.

Caneca?

O conceito de caneca abrange não apenas o ato de tomar café, inclui todo mundo desde quem bebe chá até aqueles que preferem cerveja.
